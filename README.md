# WEBTE2 Exam API

RESTful web API developed as part of the **final exam assignment** for the course  
**B-WEBTE2 – Web Technologies 2** at **FEI STU** (2024/2025 academic year).

---

## Project Overview

This repository contains a **REST API backend application** created for the B-WEBTE2 exam.  
The project demonstrates practical knowledge of web backend development, REST principles, and API design.

The API provides endpoints for working with structured data and supports standard CRUD operations using HTTP methods.
